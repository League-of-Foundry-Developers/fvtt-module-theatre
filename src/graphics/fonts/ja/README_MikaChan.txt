[About Mikachan font]
Mikachan-font is Japanese font.
It was written by Mikachan,and scaned and used font making software.

It is possible to display
Hiragana,Katakana,Numbers,Alphabets,symbols,Kanji for education,Usual kanji,
Kanji of person's name,JIS kanji Level 1,JIS kanji Level 2,IBM expantion kanji.
