戸越明朝 (Togoshi Mincho)

戸越明朝は、東風フォント代用品（OBSOLETE）明朝体をベースに、グリフ（字形）を調整して作成した TrueType フォントです。調整は FontForge を使って 1 文字ずつ手で行いました。

戸越明朝は、東風フォント代用品と異なり、アウトラインフォントだけで構成されています。ビットマップフォントは含まれていません。また、東風フォント代用品と異なり、英数字がプロポーショナルになっています。


なお、このフォントは開発途中のものであり、かつ十分に検証されているわけではありません。

フォントの構成について

東風フォント代用品に含まれる README.ja によると、東風フォント代用品の明朝体のアウトラインフォントは、下記のような構成になっています。

  仮名、英数字：従来の東風明朝
  ギリシャ文字、キリル文字：内田さん製作の拡張フォント
  それ以外：和田研明朝

戸越明朝では、仮名、英数字、漢字を中心に変更を加えています。

ギリシャ文字、キリル文字については、東風フォント代用品のものをそのまま使っています。

インストール方法と設定

戸越明朝は、通常の TrueType フォントです。特別なことは何もありませんので、通常のフォントと同様にインストールできると思います。


Linux の場合、ヒンティングをオフにしたほうが綺麗に見えると思います。設定方法はさまざまあるかと思いますが、例として戸越ゴシックだけに適用する手順を記します。この設定方法は、Fedora 8、Ubuntu 7.10 で試しました。


まず、~/.fonts.conf ファイルを作成し、下記のように記述します。

<?xml version="1.0"?>
<!DOCTYPE fontconfig SYSTEM "file:///etc/fonts/fonts.dtd">
<fontconfig>
  <match target="font">
    <test name="family">
      <string>Togoshi Mincho</string>
    </test>
    <edit name="hinting" mode="assign">
      <bool>false</bool>
    </edit>
  </match>
</fontconfig>

そして、ログインし直すなどして X を起動し直します。


戸越ゴシックなど他のフォントについても同様の設定にしたい場合、<match>要素を複数設定して対応することができます。設定の詳細については、man 等が参考にできます。man は「man fonts-conf」で起動できます（なお、man は終了は「q」、ヘルプは「h」です）。

なお、fonts.conf の設定によっては、欧文フォントとの「合成フォント」も実現可能です。戸越ゴシックの英数字では満足できない場合等ありましたら、設定で対応していただくこともできます。
